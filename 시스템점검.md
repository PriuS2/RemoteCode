## CRITICAL (즉시 수정 필요)

- **내용:** `allow_origins=["*"]` + `allow_credentials=True` 조합은 보안 안티패턴

- [ ] 수정 완료

### 4. python-jose 보안 취약점

### 7. XSS 위험 - dangerouslySetInnerHTML

- **수정:** DOMPurify 등 HTML 새니타이저 추가 또는 안전한 렌더링 방식 사용
- [ ] 수정 완료

### 8. Error Boundary 없음
- **파일:** `frontend/src/main.tsx`
